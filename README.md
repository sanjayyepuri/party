# Party




## API Endpoints
```
GET /hello
```
Endpoint that uses the OTP to display the passphrase to the guest.

```
GET /rsvp
```
Retrieve a guest's RSVP status

```
UPDATE /rsvp
```
Update a guest's RSVP status


```
GET /guests
```
Get the guest list


```
GET /party
```
Get the party information


```
POST /auth
```
Send passphrase and retrieve JWT token
